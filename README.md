## 使用webpack
初始化package.json
```
npm init -y
```
- 全局安装 (不推荐)
```
npm install webpack  -g
```


- 本地安装
```
npm install webpack webpack-cli -D  （开发模式下使用）
``` 


## 在webpack中所有文件都是模块
-js 模块  模块化（AMD CMD es6Module  commonjs）

## 直接允许webpack
会执行node_modules对应的bin下的webpack.cmd
```
npx webpack
```
